# Field Grid Logo Concepts

Third-pass FarmTrack logo directions around one stronger metaphor:
abstract farming fields as crypto protocol parcels, with a track/route across them.

- `farmtrack-field-01-parcel-route.svg` - flat field parcels with a clean route.
- `farmtrack-field-02-tilted-plots.svg` - slightly isometric, more spatial.
- `farmtrack-field-03-drop-zone.svg` - airdrop landing into a highlighted parcel.
- `farmtrack-field-04-contour-grid.svg` - map/contour feel, less literal.
- `farmtrack-field-05-command-field.svg` - terminal/tooling flavor entering the field grid.
- `farmtrack-field-06-wordmark.svg` - horizontal lockup using the parcel route sign.
