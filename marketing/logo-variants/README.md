# FarmTrack Logo Variants

Five SVG directions for the FarmTrack brand, based on the README positioning:
local desktop app, crypto airdrop farming, wallets/protocols, P&L tracking, and local-first privacy.

## Files

- `farmtrack-logo-01-growth-ledger.svg` - closest evolution of the current bar-chart icon.
- `farmtrack-logo-02-wallet-grid.svg` - wallet, protocol graph, and FT monogram.
- `farmtrack-logo-03-local-vault.svg` - local/private data vault plus tracking line.
- `farmtrack-logo-04-protocol-orbit.svg` - multi-protocol command center mark.
- `farmtrack-logo-05-terminal-wordmark.svg` - horizontal wordmark for README, splash, or marketing.
- `farmtrack-logo-preview.svg` - contact sheet with all variants.

## Notes

- Palette follows the app's warm dark OpenCode direction: `#201d1d`, `#151312`, `#3a3131`, with blue/green/amber accents.
- These are source variants only. The current app icon and UI references were not changed.
- For app icons, variants 01, 02, 03, and 04 are square. Variant 05 is a wordmark.
