# FarmTrack Brand

Primary direction: `Tilted Plots`, selected from the field-grid logo concepts.

- `farmtrack_icon.png` and `farmtrack_icon.ico` are generated from `marketing/logo-variants/field-grid/farmtrack-field-02-tilted-plots.png`.
- `static/brand/farmtrack-mark.svg` is the in-app mark.
- `static/brand/farmtrack-wordmark.svg` is the reusable horizontal lockup.
- `marketing/brand/farmtrack-brand-header.svg` is the GitHub/README header.

Use the mark for app icons and compact UI. Use the header or wordmark for GitHub, posts, screenshots, and release pages.
